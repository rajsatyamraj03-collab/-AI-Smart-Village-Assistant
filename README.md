# 🌾 AI Smart Village Assistant

## 💡 Problem
Rural communities lack easy access to:
- Farming advice
- Healthcare info
- Government schemes

## 🚀 Solution
A voice-based AI assistant using Google Gemini API.

## 🧠 Features
- 🎤 Voice Input (Hindi/English)
- 🤖 AI Responses (Gemini)
- 🔊 Voice Output
- 🌾 Farming Advice
- 🏥 Health Tips
- 📜 Govt Schemes

## 🛠 Tech Stack
- HTML, CSS, JavaScript
- Google Gemini API
- Speech Recognition API
- Firebase (optional)

## ▶️ How to Run
```bash
npm install
npm start
